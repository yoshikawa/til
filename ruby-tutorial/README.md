## Setup
1. リポジトリのクローン
   ```
   git clone https://github.com/YoshikawaTaiki/virtualmachine-template.git
   ```

1. vagrant up

   `vagrant up`

1. 仮想マシンに接続

   `vagrant ssh`

## Install
1. リポジトリのクローン
   ```
   git clone https://github.com/YoshikawaTaiki/rails-tutorial.git
   ```

1. 内部サーバの立ち上げ
   
   `sudo rails s`
